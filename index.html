Formatting Style Options

The style option you choose determines the classes applied to your generated HTML table.

  • Unstyled: Produces a plain HTML table with no CSS classes applied. Best for copying into an environment where you control the styling.
  • Tailwind CSS (Utility Classes): Adds Tailwind utility classes (e.g., `min-w-full`, `border-gray-300`) directly to the HTML elements. Requires the Tailwind CSS framework to be loaded on your final page.
  • USWDS (U.S. Web Design System): Adds USWDS-specific classes (e.g., `usa-table`, `usa-input`). Requires the USWDS framework to be loaded on your final page.
